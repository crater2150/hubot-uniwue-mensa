## Installation

In hubot project repo, run:

`npm install hubot-uniwue-mensa --save`

Then add **hubot-uniwue-mensa** to your `external-scripts.json`:

```json
[
  "hubot-uniwue-mensa"
]
```

user1>> hubot hello
